コマンド(git bush)での動かし方
個人ファイル
↓ git add ファイル名 *ファイル名を'.'にすると全てadd
ステージングエリア
↓ git commit *git commit -v:変更内容を確認しながらコミット
ローカルリポジトリ
↓ git push ↑ git pull
リモートリポジトリ

1. git commitでvimエディタへ移動
2. i:挿入モード
3. コミットメッセージを入力
4. escキーでノーマルモード
5. :wq:メッセージを保存してエディタを閉じる

* commitをまとめて行うことはできるが，メッセージが一つしか設定できなくなるのでひとつづつcommitするべし
* git status:どのファイルが変更されたかを表示
* git log:コミット履歴を確認
    git log --one line:コミット履歴を一行で表示
    git log -p ファイル名:ファイルの差分を表示
    git log -n 数字:最新の数字回のコミット履歴を表示 -> j:下へ k:上へ q:終了
* git diff:ステージングエリアとローカルリポジトリとの差分を表示
* git reset:直前のaddやcommitを消す
* コマンド集
    cd:移動 ls:内容確認 ls -a:隠しファイルまですべて mkdir:新規作成 re:削除 cp:コピー mv:ファイル移動と名前変更
